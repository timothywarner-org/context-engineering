# Utility Scripts

Helper scripts for validation and maintenance.

## Contents

| Script | Description | Usage |
|--------|-------------|-------|
| `validate-environment.js` | Verify your development setup | `node validate-environment.js` |
| `validate-links.js` | Check for broken links in markdown | `node validate-links.js` |
| `check-markdown-spacing.js` | Lint markdown formatting | `node check-markdown-spacing.js` |
| `fix-markdown.js` | Auto-fix markdown issues | `node fix-markdown.js` |

## Environment Validation

Run this before class to ensure your setup is complete:

```bash
cd /path/to/context-engineering
node scripts/validate-environment.js
```
