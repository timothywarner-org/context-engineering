Minimization

On write, content is compressed by an LLM to preserve meaning with fewer tokens. This reduces context window usage in downstream reasoning.

Before (12 tokens):
"WRN-00006 has thermal issues when running hydraulics"
After (8 tokens):
"WRN-00006: thermal issues with hydraulics"

Enrichment

On read with enrich=true, minimized content is expanded by an LLM to provide richer context. Results are cached per entry.

Stored:
"WRN-00006: thermal issues with hydraulics"
Enriched:
"WRN-00006 experiences thermal regulation problems when the hydraulic system is active..."

LangGraph Integration

Scratchpad entries are automatically injected into the RAG pipeline via the inject_scratchpad node, appearing before graph context.

1. Parse → 2. Graph → 3. Scratchpad → 4. Retrieve → ...
